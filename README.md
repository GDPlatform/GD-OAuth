# GDPlatform - OAuth (Geode Mod)

GD-OAuth is a Mod used to work Authentication Code and API.

For Example (JavaScript):
```js
fetch("https://api--gdplatform.netlify.app/oauth/code", {
  method: "POST",
  body: JSON.stringify({
    code: "ABC123"
  })
});
```

[**See GDPlatform Documentations about how it works with Auth**](https://gdplatform.netlfiy.app/docs)
